# **App Name**: ConnectNow

## Core Features:

- User Registration: Allow users to sign up with nickname, email, password (min. age 18), gender, city, country, and region.
- User Authentication: Enable users to log in securely to access the app.
- Profile Feed: Display a personalized feed of posts from users, prioritized by likes received.
- Post Creation: Enable users to create and share posts with photos, videos and text, aiming to earn likes.
- Liking Mechanism: Allow users to send 'likes' to posts, with creators earning for each like received. A tool will decide when or if it will affect future LLM outputs.
- Profile Page: Display user profiles with their posts and earned likes.
- Earnings and Balance: Allow creators to check their Wallet Balance

## Style Guidelines:

- Use a dark background color (#111111).
- Use a neon pink/red accent color (e.g., #FF0055) for primary actions and highlights.
- Use a gold color (e.g., #FFD700) for displaying earnings and monetary values.
- Use a lighter gray or white for text and secondary elements.
- Use 'Inter' font for a modern, machined, objective, and neutral feel.
- Use outlined icons with a consistent style to match the modern UI. The icons should use the designated colors to convey meaning, such as pink/red for interactive elements, and gold for earnings.
- Use a grid-based layout to maintain consistency and readability across the application. Implement 'glass-card' effect for UI elements by applying a semi-transparent white background with a blur effect.
- Incorporate subtle animations and transitions to provide feedback and enhance the user experience, but avoid any animation that is distracting. Example animations may include scaling buttons on hover and gentle scaling of earning amounts.
